# cohort-retention-analysis
- SQL-запити: cohort-retention-analysis.sql 
- Когортна таблиця: https://docs.google.com/spreadsheets/d/1VIyh-V-o1nWiSVo1k9yWAfPGB2OwxcBd3LJZFH6TEWs/edit?usp=sharing 


Інструменти: SQL, Google Sheets 

Опис: Проводила когортний аналіз для оцінки утримання користувачів (Retention Rate) двох сегментів аудиторії – промо та органічних користувачів, виявляючи відмінності в поведінці та якості залучення для підтримки стратегії зростання бізнесу.

Мої завдання: 
- Підготовка та трансформація даних.
- Розрахунок та сегментація користувачів за когортами.
- Побудова інтерактивних когортних таблиць у Google Sheets. 
- Порівняльний аналіз Retention Rate між сегментами.

Результат: Виявлено суттєві відмінності в утриманні між промо та органічними користувачами. Надано рекомендації щодо оптимізації стратегії залучення.
